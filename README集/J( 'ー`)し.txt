■パッチ概要
　・書店か自室で勉強出来る様変更
　・円光料金がランダムで変わるよう変更
　・円光発生時に持ち合わせなくてもエロエロできる問題の修正
　・NTR発展ではない円光イベントの発生オプション追加
　　⇒裏通りまたはどこでも発生を選択できます
　・円光中またはソープ中は怒って帰ることが無いよう変更
　・倍額以上払うことで一時従順ボーナス追加
　・ホテルでなくともそこの物陰でうっへっへ
　・細かなオプションの追加
