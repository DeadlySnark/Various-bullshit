主に、喫茶店でお茶を注文できるようにするパッチです。
eraIm@scgpro0.52に、アイテム説明.ERBとeraIm@scgpro0.53を咥えこませた上で
フォルダごと上書きしてください。
移動制限パッチも、当パッチに同梱しています。


お茶を注文する
	・喫茶店でコマンドお茶を淹れるを使用できるように。
	　喫茶店にいる場合、コマンド名をお茶を注文するに変更。地の文を追加。

相性の設定
	・アイドル間の相性をユニットごとに設定しました。
	　設定したのは下記のユニット、相性数値はとりあえず一律して150に。
		あんきら
		インディヴィジュアルズ
		カワイイボクと142's
		ダークイルミネイト
		トライアドプリムス
		ニュージェネレーション
		ヒーローヴァーサス
		ロック・ザ・ビート
	・また、佐久間まゆからあなたを200に設定、ついでに身体設定を追加。

不具合修正
	・なりきり時、アイドルからコピーした娼婦紋を削除するように。
	・渋谷凛の相性に本田未央だけがいない意味深な事態を修正。
	　多田李衣菜の相性に栗原ネネがいたのを修正。
	・結城晴の服装設定を修正。しかし着ない。

既知の不具合
	・あなた以外への相性が反映されない。
