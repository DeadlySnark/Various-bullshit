
ざっくりと自分が欲しかったパッチを作ってみました
放課後仮定パッチ＋修正パッチを導入した後、試したいパッチに書いてあるファイルを上書きしてください

・通報するコマンドのデバッグ
　場所を問わずキャラ番号の若い順にエネミー削除→同じ場所のエネミーをキャラ番号の若い順に削除するように
	COMF408.ERB

・会話しかすること無いのに同一コマンドって言われても困るんだよパッチ
　元の紅魔館とかだと掃除とかできたんですけどねぇ
	SOURCE.ERB

・隣に誰が居るか見えちゃうパッチ
　覗くコマンドとかぶるけどそっちはオナニーとか見えるみたいだから別にいいよね！なぜかこっちは弱み握れないっぽいけど！
　NTRオプションから、居場所筒抜けモードをONにする必要があります
	COMF400.ERB

・特訓中なんだから勝手に移動すんなパッチ
　同行指定した上で特訓した場合、最低でも次のコマンドを選ぶまでは同行指定が保持されます
	MOVEMENT.ERB


・拉致率高すぎね？パッチ
　キャラ追加から問答無用でNTRれるまでに猶予期間を設定できるようにしました
　本名を知らない間に一定期間（選択可）が過ぎるとNTRられます
　猶予期間を3日にすると名前を教えてもらえる難易度がちょっと上がります
　猶予期間を設定した場合、期間内に間男を全部タイーホできた場合もNTRを防ぐことが出来ます
　猶予期間を0日にした場合、大体今までと同じ挙動を取るはずです

　アイドルの初期位置（＝帰宅場所）と間男の初期位置とその場から立ち去った場所その他諸々が全部場所_ダミーだったため
　起床後ものっそい勢いで声掛けられていたので、初期位置をそれぞれ別の場所に指定しました
　アイドルの最大人数が青天井だったので、念のため200人までにしときました（プレイヤー＋アイドル＋オッサンで最大211人）
　オッサン強化オプション使用時に理論上無限にオッサンが湧く可能性があったので、最大１０人までにしました
	DIM.ERB EVENTTURNEND.ERB EBENTCOMEND.ERB NTR_OPTION.ERB 登録操作.ERB imas.erb COMMON_PLACE.ERB

ちなみに、オッサンは強化されてなければキャラ追加タイミングで一旦削除され、最大３人まで補充されます
