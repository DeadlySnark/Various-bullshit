主に、学生キャラの登下校徘徊を防止するパッチです。

喫茶店でお茶パッチ、移動制限修正パッチ、無理矢理適当色づけパッチ、居酒屋パッチ＋αの各種パッチを導入した上で
当パッチをフォルダごと上書きしてください。


パッチ内容

	繁華街→学校→繁華街を繰り返すのを暫定防止。
		15時以降は放課後として、学校に行かないように
			あくまで登校しないだけなので、下校はばらつきます
		15時になるまでは徘徊するが、これを制限するとバグとして位置番号0に飛ばされるので保留
	下校時間の目安ができるので、多少は学生攻略がやりやすくなると思います。

	活動時間を使用されていない箇所に記述していたようなので修正。
		小○生を6半時から20時
		中○生を7時から21時
		高○生を7時から22時
		その他を7時半から24時に
	同行中のキャラクターは同行解除するまで帰宅、就寝をしません。

	公園トイレにいる場合、押し倒すコマンド名称が個室に連れ込むになるように。
		情事目撃される時に変な感じかも知れない

	一定年齢以上の駄菓子屋への移動制限を廃止しました
