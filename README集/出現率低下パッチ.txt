eraIm@scgpro v0.41用　出現率低下オプション追加ぱっち

○対象環境
　eraIm@scgpro v0.40＋v0.41

　ご自由にお使いください。
　使用前のセーブは取っておいてください。何かあってもアプデ前に低下フラグを全員解除すれば問題ないとは思いますが。



○作業内容

・出現確率制御.ERB
　出現率:キャラNOが 1で出現率上昇, 2で出現率低下になるように。
　あと全反転の処理に悩んで削除。全上昇、全抑制、全解除の3つに。
＜製作者向け＞
　出現率はbit管理する予定だったのでしょうか？
　増加と抑制が同時でも困るし一々両方付け消しするのも面倒であまりbit要素残ってないです、申し訳ない。

　出現率:0はちょっとややこしくなりました。
　増加してるアイドルがいればSETBIT 出現率:0, 0、低下でSETBIT 出現率:0, 1されます。0~3。
　低下だけしている時に、増加中の基本49/50再抽選処理が来ると処理が重そうに見えたので、つい。



・OPTION.ERB
　上記仕様に合わせた。


・登録操作.ERB
　抑制中処理を追加。
　抑制中キャラは出現率が1/50になります。
　完全に抑制しないのは、出現可能アイドルが20~30未満ぐらいで無限ループしそうな気がするので、その予防です。
　確率を変える場合、47~52行あたりを適当に変えてください。


・COMF405.ERB（電話する）
　非出現かつ出現率低下中のキャラが表示されないように。
　出現率とあんまり関係ないですが、日数増えると表示欄増えすぎてよく分からなくなるので。
