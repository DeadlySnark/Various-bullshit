スレで話題になってた噴乳関係のパッチを自分用に作ってみたけどせっかくなので公開



・お薬追加
母乳体質を付けたり消したりするいつものお薬を通販に追加します（要[禁断の知識]）
自分の確認用に作りましたが後付で付けたり消したりしたい時にどうぞ


・噴乳抑制（仮）
普段噴乳しないように絶頂かある程度の快Ｂ（1ターンで+3000以上）をトリガーにします
母乳を溜める内部計算の影響で最初のうちはＢ絶頂以外噴きづらくなりますが仕様です
性経験が増えていって各部位の感度が上がるとＢ以外の絶頂や胸愛撫程度で噴乳するようになります


・噴乳抑制（仮）怒っちゃやーよ版
上記のトリガーに加え「恋慕または親愛持ち」かつ「押し倒しまたは押し倒され状態」の時の噴乳は不潔のソースが半減し怒りにくくなります
「同意の上で押し倒したor向こうから押し倒してきたのに噴乳したら怒って帰った」とかアレなので･･･ｗ
（あくまで半減なので怒ることもあります）
