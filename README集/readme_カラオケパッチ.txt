eraIm@scgpro v0.51用　カラオケパッチ

＜対象環境＞
eraIm@scgpro v0.51

＜内容＞
・カラオケにいる時、「カラオケを利用する」が選択できます。
　選択すると、料金を払って最大3時間「カラオケ(個室)」に移動できます。
・個室では「曲を入れる」が選択できます。
　アイドルを連れている場合、アイドルも一緒に歌います。
　歌ったアイドルは、僅かにボーカル経験が得られます。
・簡易ラブホとしても利用可能です。
・うふふ状態でも歌えます。

＜注意事項＞
・改変・再配布等は自由です。
