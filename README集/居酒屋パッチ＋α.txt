■居酒屋実装
・以下の条件を満たす場合、「お酒を注文する」が選択できます。
　- 同行者が居ない
　- 現在地が居酒屋
・18〜23時の間に「お酒を注文する」を選択すると、
　2/3の確率で、20歳以上のキャラクターがランダムに出現します。
・未出現キャラも対象に含まれます。（地の文がやや変わります）
　ただし、声を掛けなければ追加はされません。
・永遠の17歳も対象に含まれます。
・スカウト済、退場済アイドルは対象外です。
・選んだお酒の種類によって、特定のキャラクターが出現しやすくなります。

■カラオケ修正
