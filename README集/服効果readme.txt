
>>529修正パッチ
eraIm@scgpro0.33にて>>529を修正するパッチです。
作者さんとは別人です。

>>529を調べたところ、
コスによって付与された効果が解除されなくなってしまうという不具合でしたので、
それを修正しました。


●やったこと
@CLOTHE_EFFECT(ARG)で行われていた処理を
経験増加、効果付与、効果解除に分割し、
経験増加を覗いた効果付与と解除を既存の呼び出し場所以外に
現象を解決するのに必要なタイミングで呼び出すようにしました。

既存の状態では付与されたコス効果を解除する為には
一部を除いてそのコスを着ていないといけない状態だったため
効果が付与された際に変数に値を入力、
解除の際はコスを着ているか否かではなくそちらを参照するようにしました。
変数は魔法少女、触手服で使われていた「CFLAG」の「コスプレ着用」を使用しました。

自慰中に820で服を脱いでも効果が継続するのは仕様と判断したので弄ってません。
